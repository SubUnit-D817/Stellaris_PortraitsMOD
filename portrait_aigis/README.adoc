== 1.このリポジトリについて
拙作のStellaris用ポートレイトMODテンプレート（species_portrait_template）を使用して作成したポートレイトMOD置き場です。 +
ポートレイト画像は生成AIを用いて作成しているので、Steam Workshopでの公開は行いません。 +
 
== 2.使い方
* zipファイルをダウンロード +
* ダウンロードしたzipファイルを解凍し、StellarisのMODフォルダにぶち込んでください。 +
